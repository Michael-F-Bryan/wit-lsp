Enum tower_lsp::jsonrpc::ErrorCode

source · 
pub enum ErrorCode {
    ParseError,
    InvalidRequest,
    MethodNotFound,
    InvalidParams,
    InternalError,
    ServerError(i64),
    RequestCancelled,
    ContentModified,
}
Expand description

A list of numeric error codes used in JSON-RPC responses.

Variants§

§

ParseError

Invalid JSON was received by the server.

§

InvalidRequest

The JSON sent is not a valid Request object.

§

MethodNotFound

The method does not exist / is not available.

§

InvalidParams

Invalid method parameter(s).

§

InternalError

Internal JSON-RPC error.

§

ServerError(i64)

Reserved for implementation-defined server errors.

§

RequestCancelled

The request was cancelled by the client.

§Compatibility

This error code is defined by the Language Server Protocol.

§

ContentModified

The request was invalidated by another incoming request.

§Compatibility

This error code is specific to the Language Server Protocol.
